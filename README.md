# Planora App

אפליקציית תכנון וניהול אירועים.

## הפעלת האפליקציה

```bash
npm install
npm run dev
```

## בניית האפליקציה

```bash
npm run build
```

## הגדרות סביבה

צור קובץ `.env` בתיקיית השורש עם המשתנים הבאים:

```
VITE_OPENAI_API_KEY=sk-your-openai-api-key
```

## טכנולוגיות

- **Frontend**: React + Vite
- **UI**: Tailwind CSS + Radix UI
- **Backend**: InstaBack
- **AI**: OpenAI API
